Schema Wizard: Upload Sample(s) Step

The Create New Schema Wizard steps the user through the following tasks:

  • Upload one or more sample files containing representative data
  • Analyze sample files on the server which discover the following:
    • file format type,
    • names of fields, and
    • characterizations for fields
  • Verify the quality of the sample data
  • Propose a candidate for the schema
  • Allow the user to modify the candidate schema
  • Save the schema
