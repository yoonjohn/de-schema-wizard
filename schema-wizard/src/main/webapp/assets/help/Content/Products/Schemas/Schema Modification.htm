Schema Modification

A request to modify a schema may be initiated while a schema is being browsed. Modification requests place the schema back into the schema creation workflow. New data sample files get uploaded or data streams are opened which undergo analysis and the results are compared with the existing schema. The data engineer has the opportunity to interact with the findings, make modifications and chooses whether to save the changes.
