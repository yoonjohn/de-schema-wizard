Catalog

The DigitalEdge Schema Wizard maintains a catalog of schemas and data sample files used in their generation. Both schemas and data sample files may be browsed. Statistical information collected during data sample file analysis is available for each data sample file. The statistical information for a schema is a merger of the statistical information of each data sample file used in the schema’s generation. Raw data for data sample files are not maintained.

 

Metadata is available for each schema which includes its version and descriptive text which the data engineer entered when the schema was initially saved. Versioning is maintained since schemas may be modified.
