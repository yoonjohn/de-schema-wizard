Create a Domain

The creation of a Domain is one of the most important parts of Schema Wizard.

A domain can range from a variety of specific areas of knowledge. For example, Aerospace, Health, Transportation, etc.

The creation of a Domain allows you to create interpretations on that specific domain.

Steps for Creation of a Domain

  1. Click the "Create New Domain" button
  2. Enter in a name and Description for the Domain.
  3. Once completed, click the "Save" button
  4. Congratulations, you have created your first Domain!
