Welcome to Schema Wizard

The DigitalEdge Schema Wizard is one of the tools in the DigitalEdge 3.0 Suite. It is used by the data engineer to create or modify schemas. A DigitalEdge schema is intended to be a unified schema for an enterprise system. Such a schema is the result of coalescing two or more schemas from different databases or data streams. Schemas are used by other tools in the suite to include the ingest feature of DigitalEdge and the VizWiz tool.

 

Schema Wizard creates a schema by a discovery process while analyzing sample data. The data may be structured or unstructured and its content may come in various forms such as: comma separated values, JSON, XML, name/value pairs, plain text, and documents. Additionally, multiple instances of the data may come in compressed file formats. The data engineer can collect the samples as files prior to using Schema Wizard or may direct live streams into a Schema Wizard session.
